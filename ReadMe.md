## Bonjour apprenants d'Alyra

Ceci est un **template** pour vous aider à vous organiser lors de l'apprentissage du développement blockchain chez Alyra  

Nous vous conseillons de cloner sur votre machine ce repo pour pouvoir travailler tranquillement dessus.  
Vous pouvez rendre vos projets directement dessus (c'est surement le plus simple), ou bien sur des repo individuels, à vous de voir ce que vous préférez.  
Le fork doit etre privé, et vous pouvez ajouter lecascyril en tant que membre du repo au moins en lecture.

*Attention: ceci est juste un template fait de dossier, il ne sera pas mis à jour avec du code. Pour le code de votre promo, veuillez pull le repo de promo :)*

### Bon apprentissage!
